# 🔥 Viral Allure Bot — Gid Konplè

## Etap 1 — Jwenn Groq API Key (GRATIS)
1. Ale sou: https://console.groq.com
2. Kreye yon kont gratis
3. Klike "Create API Key"
4. Kopye kle a

## Etap 2 — Deplwaye sou Vercel (GRATIS)
1. Ale sou: https://vercel.com
2. Konekte ak GitHub ou
3. Enpòte dosye sa a
4. Ajoute variables yo:
   - APP_SECRET = kle sekrè Facebook ou
   - PAGE_ACCESS_TOKEN = token paj ou
   - GROQ_API_KEY = kle Groq ou
   - VERIFY_TOKEN = viral_allure_webhook_2026

## Etap 3 — Konfigire Webhook Facebook
1. Ale sou developers.facebook.com
2. Klike "Messenger" nan meni gòch
3. Ajoute Webhook URL: https://[URL-VERCEL-OU]/webhook
4. Verify Token: viral_allure_webhook_2026
5. Koche: "feed" nan subscriptions
